# 🛡️ HyperTask - Task Management App

**HyperTask** هو تطبيق متكامل لإدارة المهام اليومية، يجمع بين التصميم العصري (UI/UX) والأداء القوي، مصمم لمساعدة المستخدمين على تنظيم يومهم بفعالية مع لمسة جمالية.

## ✨ المميزات (Features)
* 📱 **تصميم عصري ومتجاوب:** واجهة مستخدم احترافية .
* 🔐 **نظام تسجيل دخول:** تأمين البيانات عبر API ونظام SharedPreferences لحفظ الجلسة.
* 💾 **قاعدة بيانات SQL:** تخزين المهام محلياً لضمان السرعة والعمل بدون إنترنت.
* 📊 **مؤشر إنجاز:** متابعة نسبة إنجاز المهام اليومية بشكل مرئي.
* 🎨 **تجربة مستخدم سلسة:** استخدام Animations و Custom Widgets مخصصة.

## 🛠️ الأدوات المستخدمة (Tech Stack)
* **Framework:** [Flutter](https://flutter.dev)
* **Backend API:** PHP / MySQL 
* **State Management:** cubit
* **Storage:** SharedPreferences
* **UI Libraries:** Awesome Dialog, Intl, Gap  cupertino_icons,lottie,intl,percent_indicator,awesome_dialog,shared_preferences ,flutter_bloc
